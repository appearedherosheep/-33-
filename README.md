3. 보안 에러가 뜰텐데 무시하고 실행해주세요 안전합니다^^
4. 오늘의 급식을 확인하세요!
5. exe 파일을 드레그해서 작업표시줄 위에 놓으면 작업표시줄에 추가 됩니다
